שפות שהשתמשתי בהן באתר


AngularJS

אנגולר נכתבה על ידי גוגל. היא ספריית MVC בצד הלקוח. היא כוללת מודל, View, Controller ו Services.

HTML 5

שפת תגיות לתצוגה ועיצוב דפי אינטרנט ותוכן לתצוגה בדפדפן. זו שפת התיכנות המרכזית בעולם האינטרנט, המהווה שלד למרבית עמודי התוכן באינטרנט. השפה מאפשרת עיצוב תוכן בצורה מהירה, קלה ללימוד באופן יחסי וקלה לכתיבה.

JavaScript

שפת תסריט מפורשת מבוססת אובייקטים המותאמת לשילוב באתרי אינטרנט ורצה על ידי דפדפן האינטרנט בצד הלקוח. השפה מרחיבה את יכולות שפת התגיות הבסיסית HTML ומאפשרת בכך ליצור יישומי אינטרנט מתוחכמים יותר.

CSS 3

CSS נוצר במטרה להפריד בין תוכן ומבנה דפי האינטרנט לבין עיצובם: עד ליצירת ה-CSS בידי קבוצת התקינה W3C ב-1995, נכתבו תוכן האתרים וסגנון העיצוב שלהם באותו דף HTML. כתוצאה מכך, הפך קוד ה-HTML למסובך ובלתי קריא, ושינויים עיצוביים באתר שלם דרשו מעבר דף אחר דף. באמצעות CSS ניתן למקם הגדרות עיצוב בקובץ יחיד, ששינוי בו ישתקף בבת אחת בכל הדפים העושים בו שימוש.

Parse

Parse היא ספריית javascript המחליפה צד שרת ודאטא בייס. Parse נותן למשתמש גישה לדאטא בייס באינטרנט בפשטות ונוחות. Parse משלבת את הServer Side לתוך הClient Side ובכך חוסכת למתכתים הרבה עבודה

WebStorm

WebStorm היא סביבת עבודה ופיתוח המיועדת לבניית אתרים. סביבת העבודה תומכת בשפות החדשות ביותר ובפלאגינים. היא פשוטה, נוחה ומתאימה את עצמה למשתמש ולשפה בה הוא משתמש.

MaterializeCSS

MaterializeCSS היא ספריית CSS המאפשרת כלים פשוטים לבניית אתרים עם מטיריאל דיזייגן

Animate.css

ספריית css המאפשרת ליצור ולהשתמש באנימציות כניסה ויציאה בצורה פשוטה ונוחה
